STAT 202 Notes

Variance: $E(x^2) - (E(x))^2$ => $var(x) = \sum_i{((y_i-\mu)^2\times P(Y=y_i))}$ or $var(x) = \sum_i{(y_i^2\times P(Y=y_i))} - \mu^2$

$E(ax+by+c) = aE(x) + bE(y) + c$
$Var(ax+by+c) = |a^2Var(x)| + |b^2Var(y)|$

Binomial distribution: $Var(Bin()) = np(1-p)$

$_nC_j = \frac{n!}{(n-j)!(j)!}$

x successes over n tries with probability p each: $P_{BIN} (_nC_x) p^x (1-p)^{n-x}$

PDF: $f(y) = P(Y = y)$

CDF: $F(y) = P(Y \leq y); f(y) = F(y) - F(y-1)$

Z = \frac{Y-\mu}{\sigma}

95th percentile: $P(Z\lt z_{1-0.95}) = 0.95$; look up 0.9500 inside Z table (or find the closest one)

Using z scores in abnormal situations: $z_{0.05} = 1.645 = \frac{y-\mu}{\sigma}$

Sampling: $\bar{y} = \frac{1}{n}\sum_i{y_i}$

Sample proportion: $\hat{P} = \frac{x}{n}$

E(\hat{P}) = E(\frac{x}{n}) = \frac{1}{n}E(x)
Var(\hat{P}) = Var(\frac{x}{n}) = \frac{1}{n^2}Var(x)
\sigma_{sample} = \frac{\sigma_{pop^n}}{\sqrt{n}}

Sample distribution goes through normal distribution of $N(\mu, \frac{\sigma_{pop^n}}{\sqrt{n}})$

mean(\bar{Y}) = E(\bar{Y}) = \mu

Law of large numbers: as $n \rightarrow \inf,\ \bar{y} \rightarrow \mu$

Binomial approximation: $P_{BIN}(X = x) = P_{NORMAL}(X + 0.5 < x) - P_{NORMAL}(X - 0.5 < x)$
