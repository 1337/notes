Integrating with an integration factor, $R(x)$

$y' + \frac{1}{2}\ y = \frac{1}{2}e^{\frac{t}{3}}$

Multiply both sides by a factor, $R(x)$.

$R\ y' + \frac{1}{2}R\ y = \frac{1}{2}R\ e^{\frac{t}{3}}$

Notice that $(Ry)' = R\ y' + R'\ y$ via product rule. We try to match the LHS with $R\ y' + R'\ y$.

$R\ y' + R'\ y = R\ y' + \frac{1}{2}R\ y$

... and we found that

$R' = \frac{1}{2}R$

Now use your 1337 separable equations skills to find R from R'.

$\frac{dR}{dt}\frac{1}{R} = \frac{1}{2}$
$\frac{1}{R} dR = \frac{1}{2} dt$
$\ln {|R|} = \frac{1}{2} t + c$
$R = e^{\frac{1}{2} t + c}$
$R = e^{\frac{1}{2} t}e^c$
$R = ce^{\frac{1}{2} t} = e^{\frac{1}{2} t} (c = 1)$

The idea here is to put R back into the modified equation... $(Ry)'$

$(Ry)' = R\ y' + R'\ y = R \times \frac{1}{2}e^{\frac{t}{3}}$

We know R and we have y'...

$((e^{\frac{1}{2} t}) y)' = e^{\frac{1}{2} t}\frac{1}{2}e^{\frac{t}{3}} = \frac{1}{2} e^{\frac{5t}{6}}$

Integrate both sides

$(e^{\frac{1}{2} t}) y = \int \frac{1}{2} e^{\frac{5t}{6}} dt = \frac{6}{10} e^{\frac{5t}{6}} + c$

Remember the $+c$.

$y = \frac{6}{10} \frac{e^{\frac{5t}{6}}}{e^{\frac{1}{2} t}} + \frac{c}{e^{\frac{1}{2} t}}$

Simplify

$y = \frac{3}{5} e^{\frac{t}{3}} + c\ e^{\frac{-t}{2}}$

Finding out if an equation is exact or not

You were given a function, $M(x,y)dx + N(x,y)dy = 0$. The differential equation is exact if

$\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$

If they are the same, you have to solve the equation. Let $G$ be the solution. $\frac{\partial G}{\partial x} = M$, and $\frac{\partial G}{\partial y} = N$. That means $G = \int M \partial x$ and $G = \int N \partial y$.

Example

$(2xy^2+2y)dx+(2x^2y+2x)y'=0$

Let $M = 2xy^2+2y$, $N=2x^2y+2x$

$\partial M / \partial y = 4xy+2$
$\partial N / \partial x = 4xy+2$

... so the DE is exact! Integrate M get G

$G = \int_x M\ dx + h(y) = x^2y^2 + 2xy + h(y)$

Differentiate that to get $dG/dy$, which was said to be N:

$dG/dy = 2x^2y + 2x$

Result matches $N$ precisely, so $h(y)$ must be a constant.

Therefore, the solution to this problem is $2x^2y + 2x + C = 0$

Second-order ODEs

Homogeneous ones with constant coefficients

In the form $1y'' + Py' + Qy = 0$, solve by trying $y=e^{mx}$

$y' = me^{mx}$ $y'' = m^2e^{mx}$ $1y'' + Py' + Qy = m^2e^{mx} + Pme^{mx} + Qe^{mx} = 0$ $e^{mx} (m^2 + Pm + Q) = 0$

Since $e^{mx}$ can never be 0, the quadratic must be 0. Solve the quadratic.

If you get two values of $m$:

Congratulations! Your solution is $y_{[c]} = Ae^{m_1x} + Be^{m_2x}$.

If you get only one value of $m$:

Try multiplying $m$ by $x$ or $x^2$ to get $m_2$.

Reduction of Order

Given one known solution, the method of Reduction of Order allows you to find a second one.

$y_2 = v(x) \times y_1$
Method of undermined coefficients (i.e. guessing)

You get a non-homogeneous equation, and you know that LHS = RHS. If you think $e^{mx}$ can do the job, guess $>Ae^{mx}$. If the RHS is cos or sin, try $A\cos x + B\sin x$.

Taylor series solutions

Differentiating a Taylor series

Say you have

$y = \sum^\infty_{n=0} a_n x^n$
$y' = \sum^\infty_{n=0} (n)\ a_n x^{n-1}$
$y'' = \sum^\infty_{n=0} (n)\ (n-1)\ a_n x^{n-2}$

But since $(n)(n-1)$ produces 0 for both $n=0$ and $n=1$, we can shift the series up by 2 without worries:

$y'' = \sum^\infty_{n=2} (n)\ (n-1)\ a_n x^{n-2}$

Shifting index of summation

$y'' = \sum^\infty_{n=2} (n)\ (n-1)\ a_n x^{n-2}$

Let $p = n - 2$ (also means $n = p + 2$) and sub p into the equation

$y'' = \sum^\infty_{(p + 2)=2} (p + 2)\ ((p + 2)-1)\ a_{(p + 2)} x^{(p + 2)-2}$
$y'' = \sum^\infty_{p = 0} (p+2)\ (p+1)\ a_{p+2} x^p$

p is a dummy variable, so simply replace p by n.

$y'' = \sum^\infty_{n = 0} (n+2)\ (n+1)\ a_{n+2} x^n$

Extracting terms from a series

Calculate the first term ($a_0$).

$y''(n = 0) = (0+2)\ (0+1)\ a_{0+2} x^0 = 2a_2$

Now that you have that term, increment your series summation by 1.

$y''= 2a_2 + \sum^\infty_{n = 1} (n+2)\ (n+1)\ a_{n+2} x^n$
