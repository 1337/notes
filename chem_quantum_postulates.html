Quantum Mechanics: postulates

  1. The state of a quantum mechanical system is completely specified by a function $\Psi$.
  2. To every observable in classical mechanics (e.g. $p$) there corresponds a linear, Hermitian operator (e.g. $\hat{P}$) in quantum mechanics.
  3. In any measurement of the observable associated with operator $\hat{P}$, the only values that will ever be observed are the eigenvalues $p$, which satisfy the eigenvalue equation $\hat{P}\Psi = p \Psi$
  4. If a system is in a state described by a normalized wave function $\Psi$, then the average value of the observable corresponding to $\hat{P}$ is given by $\lt \Psi | P | \Psi \gt$.
